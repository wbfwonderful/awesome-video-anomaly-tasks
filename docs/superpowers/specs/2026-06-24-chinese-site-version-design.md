# Chinese Site Version Design

## Goal

Add a Chinese version of the public documentation and website while reusing the existing structured data, styles, and rendering logic.

## Scope

- Keep `README.md` as the English README.
- Add `README.zh-CN.md` as the Chinese README.
- Keep all `data/` files shared between languages.
- Keep English pages at their current paths.
- Add Chinese HTML shell pages under `zh/`:
  - `zh/index.html`
  - `zh/papers/index.html`
  - `zh/datasets/index.html`
  - `zh/leaderboards/index.html`
  - `zh/leaderboards/dataset.html`
- Add language links so each English page can jump to its matching Chinese page and each Chinese page can jump back to English.

## Architecture

The Chinese pages are independent static HTML shell files. They contain Chinese document titles, headings, subtitles, table headers, navigation labels, placeholders, and initial fallback content. They load the same CSS, JavaScript, CDN dependency, and `data/` files as the English pages.

Runtime-rendered text moves behind a lightweight shared i18n helper. Each page reads its language from `<html lang="...">`; English pages use `en`, Chinese pages use `zh-CN`. The helper provides UI labels for status messages, filter labels, table labels generated by JavaScript, link labels, provenance labels, and empty states. Data values such as paper titles, method names, dataset names, venues, track names, metric keys, URLs, and tags remain unchanged.

## Language Switching

Each page exposes a visible language switch in the header navigation:

- English home: `中文` links to `zh/`.
- Chinese home: `English` links to `../`.
- English papers/datasets/leaderboards indexes link to their `zh/...` counterpart.
- Chinese papers/datasets/leaderboards indexes link back to their English counterpart.
- Dataset leaderboard links preserve the `dataset` query string when switching languages.

The language switch is plain navigation, not in-place rerendering. This keeps the README approach and website approach consistent.

## README

`README.md` and `README.zh-CN.md` link to each other at the top:

```markdown
[English](README.md) | [中文](README.zh-CN.md)
```

The Chinese README mirrors the current English README content and keeps code examples, file paths, YAML keys, command lines, repository names, and URLs unchanged.

## Testing

Tests should cover:

- Chinese HTML files exist and load shared assets with correct relative paths.
- English pages include links to the matching Chinese pages.
- Chinese pages include links back to the matching English pages.
- Dynamic JavaScript labels can render in Chinese based on `<html lang="zh-CN">`.
- Dataset leaderboard language switching preserves the current `dataset` query string.
- Existing data schema and leaderboard model behavior remain unchanged.

## Out Of Scope

- Translating structured data records in `data/`.
- Adding automatic browser-language detection.
- Adding URL query based language switching.
- Maintaining two copies of rendering logic.
